# Kindle Highlights

[![GitHub Pages](https://img.shields.io/badge/GitHub-Pages-blue?logo=github)](https://madebytinystudio.github.io/kindle-highlights)

## Introduction

Kindle Highlights is a simple and efficient web tool designed to help you parse your Kindle highlights from the `My Clippings.txt` file. Easily organize and export your highlights into Markdown format or copy them directly to your clipboard.

## Features

- Drag and drop your `My Clippings.txt` file for quick parsing
- Export highlights to clean, well-formatted Markdown
- Copy highlights to clipboard with one click
- Lightweight and easy to use in any modern browser
- No installation or account required

## Usage

1. Open the web tool in your browser.
2. Drag and drop your `My Clippings.txt` file onto the page.
3. View your organized highlights instantly.
4. Export to Markdown or copy to clipboard using the provided buttons.

```markdown
# Example Highlight Title

> Your highlight text here...

- Location: 123-125
- Date: 2024-06-01
```

## Installation

No installation needed! Just open the web page in a modern browser.  
You can also clone this repository and open `index.html` locally:

```bash
git clone https://github.com/madebytinystudio/kindle-highlights.git
cd kindle-highlights
open index.html
```
